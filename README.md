# Projeto DrinkWater

## ?? Como rodar a aplicação

Este projeto pode ser executado de três formas:

- Localmente (produção) porta 8000

npm start

- Localmente (development) porta 3000 com banco externo

Com VSCode rodar F5 ou npm run dev

OBS: Rodar docker compose up db para subir o banco isoladamente

- Totalmente via Docker Compose para simular ambiente de produção

docker compose up
